# Homebrew Tap for ToggleTUI

This is a custom Homebrew tap for [ToggleTUI](https://github.com/falt/toggletui).

## Installation

```bash
brew tap falt/tap
brew install toggletui
```

Or in one command:

```bash
brew install falt/tap/toggletui
```

## Usage

After installation, run:

```bash
toggletui
```

On first run, you'll be prompted to enter your Toggl API token.
Get your token from: https://track.toggl.com/profile

## Updating

```bash
brew update
brew upgrade toggletui
```
